# 🐺 CodeWolf - Gamified Coding Learning Platform

🚀 **Live Demo:** https://wolf-code-quest.base44.app/

CodeWolf is a gamified interactive coding education platform inspired by language-learning applications like Duolingo. It helps users learn programming languages through structured lessons, challenges, progression systems, and motivational game mechanics.

The goal of CodeWolf is to make learning programming more engaging, consistent, and enjoyable by transforming coding education into a daily habit.

---

## ✨ Features

### 🎯 Interactive Learning System
- Step-by-step programming lessons
- Beginner to advanced learning paths
- Practice-based coding challenges
- Progressive difficulty system

### 🎮 Gamification
- XP (Experience Points) system
- Level progression
- Daily learning streaks
- Achievement-based motivation
- Life/energy system

### 💻 Supported Programming Languages

- Python
- JavaScript
- Java
- C
- C++
- C#
- PHP
- SQL

### 📚 Learning Experience

- Structured courses
- Interactive exercises
- Progress tracking
- User-focused learning flow

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/codewolf-platform.git
```

Navigate to the project folder:

```bash
cd codewolf-platform
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
